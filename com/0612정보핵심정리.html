인쇄용 미리보기

정보 교과 시험 완벽 대비 핵심 정리

Ⅰ. 시험 범위 및 핵심 (★★★)

선생님께서 **지필 시험**과 **프로그램 실습 평가** 두 가지를 본다고 하셨습니다.

  1. 지필 시험 범위: 리스트(List)까지 입니다.
  2. 프로그램 실습 평가 범위: 제어 구조(Control Structure)까지 입니다.
    • 다음 주까지 제어 구조 진도가 끝나지 않으면, 시험은 그 다음 주로 연기될 수 있습니다.

Ⅱ. 실습 문제 풀이: 용병 리스트 (★★★★★)

이 문제는 시험에 나올 확률이 매우 높습니다. 코드의 각 부분을 완벽하게 이해해야 합니다.

문제 상황: 2차원 리스트 `soldier`에 용병의 이름과 무기가 저장되어 있습니다. 사용자에게 번호를 입력받아 해당 용병의 무기를 출력하는 프로그램입니다.

# 제공된 2차원 리스트
soldier = [
    ['암살자', '광전사', '추적자', '마법사', '사냥꾼'],
    ['단검', '도끼', '활', '지팡이', '채찍']
]

1단계: 용병 목록 출력하기

목표: `print` 함수와 문자열 포매팅(`%s`)을 사용하여 용병 목록을 정해진 형식으로 출력합니다.

# 핵심 코드
print("용병은 다음과 같습니다.")
print("1. %s 2. %s 3. %s 4. %s 5. %s" % (
    soldier[0][0], # 암살자
    soldier[0][1], # 광전사
    soldier[0][2], # 추적자
    soldier[0][3], # 마법사
    soldier[0][4]  # 사냥꾼
))

⭐️ 중요 포인트: 여러 개의 값을 `%`를 이용해 포매팅할 때는 `(값1, 값2, ...)` 처럼 괄호로 묶어줘야 합니다.

2단계: 사용자에게 용병 번호 입력받기

목표: `input()` 함수로 사용자에게 번호를 입력받고, `int()` 함수로 숫자로 변환합니다.

# 핵심 코드
select_str = input("용병을 입력하세요: ")
select_num = int(select_str)

⭐️ 중요 포인트: `input()`으로 받은 값은 문자열(string)이므로, 반드시 `int()`를 사용해 정수(integer)로 변환해야 합니다.

3단계: 선택한 용병의 무기 출력하기

목표: 입력받은 번호를 이용해 `soldier` 리스트에서 해당 용병의 무기를 찾아 출력합니다.

# 핵심 코드
print("용병의 필살 무기는 %s입니다." % (soldier[1][select_num - 1]))

⭐️ 매우 중요 포인트: 사용자는 1부터 입력하지만 리스트 인덱스는 0부터 시작하므로, 입력받은 값에서 **반드시 1을 빼야**(`select_num - 1`) 정확한 위치를 찾을 수 있습니다.

Ⅲ. 코딩 시 흔히 하는 실수 및 주의사항

  • 오타(Typo): 변수 이름, 함수 이름을 잘못 쓰는 경우가 가장 흔합니다.
  • 인덱스 오류: 리스트의 범위를 벗어나거나 `+`나 `-` 계산을 잘못하는 경우입니다.
  • 문법 오류: 쉼표(,), 괄호(), 따옴표("") 등 작은 문법 실수를 주의해야 합니다.

Ⅳ. 선생님의 추가 강조: 미래 기술과 프로그래밍

  • 데이터 분석의 중요성: 이제 전공과 상관없이 데이터 분석 능력은 필수입니다.
  • 생성형 AI와 바이브 코딩: AI에게 정확히 질문하여 프로그램을 만드는 능력이 중요해졌습니다.
  • 교내 '언락(Unlock)' 프로젝트: 생성형 AI를 이용한 프로그래밍을 체험하고, 진로 활동에 특별한 경험을 더할 좋은 기회입니다.
