하드코딩 vs 입력 기반 프로그래밍

더 유연하고 효율적인 프로그램을 만들기 위한 핵심 개념입니다.

하드코딩 (Hard Coding)

코드 안에 숫자나 문자를 직접 고정시켜 넣는 방식입니다. 예를 들어, `result = 3 + 4` 처럼 값을 직접 지정하는 것이죠. 이는 프로그램이 특정 값에 종속되어 유연성이 떨어집니다. 값이 바뀔 때마다 코드를 직접 수정해야 합니다.

입력 기반 프로그래밍 (Input-based Programming)

사용자로부터 값을 입력받아 그 값에 따라 계산하거나 동작하는 방식입니다. `num1 = input("첫 번째 숫자를 입력하세요: ")` 처럼 사용자에게 값을 입력받으면, 프로그램은 더 유연해지고 다양한 상황에 대처할 수 있습니다.

"계산기 만들 건데 3+4밖에 못 해? 그럼 계산기 아니지~"

변수란 무엇인가?

변수는 데이터를 저장하는 공간이자, 마치 '이름 붙인 상자'와 같습니다.

변수: 데이터를 저장하는 '이름 붙인 상자'

프로그램에서 자주 사용하거나 변동될 수 있는 데이터를 메모리에 저장해 놓고, 그 데이터를 이름으로 불러서 사용하기 위해 변수를 사용합니다. 복잡한 메모리 주소 대신 사람이 기억하기 쉬운 '이름'으로 데이터를 관리하는 것이죠.

"이름만 기억하면 값은 자동! 변수 없으면 사람 뇌가 터져요~"

변수 이름 붙이기 규칙

  • 반드시 영문자로 시작해야 합니다.
  • 숫자는 변수 이름의 뒤에만 붙일 수 있습니다. (예: `a1`, `num2`는 가능, `1a`, `2num`은 불가능)
  • 특수문자(언더스코어 `_` 제외), 띄어쓰기는 사용할 수 없습니다.

"숫자로 시작하면? 파이썬은 계산식인 줄 안다니까~"

변수와 메모리의 관계

컴퓨터는 모든 데이터를 **메모리(RAM)**에 저장합니다. 하지만 메모리 주소는 복잡한 숫자로 이루어져 있어 사람이 외우기 어렵습니다. 변수는 이 메모리 주소에 사람이 이해하기 쉬운 '이름'을 붙여주는 역할을 합니다.

사칙연산과 연산자

데이터를 가지고 다양한 계산을 하기 위해 '연산자'를 사용합니다.

  • `+`: 더하기
  • `-`: 빼기
  • `*`: 곱하기
  • `/`: 나누기 (결과는 실수)

특수 연산자 (중요!)

  • `//`: 몫만 구함 (정수)
  • `%`: 나머지 구함
  • `**`: 제곱 (a**b = a의 b제곱)

두 값이 '크다', '작다', '같다' 등을 비교하여 참(True) 또는 거짓(False)을 반환합니다.

  • `>`: 크다
  • `<`: 작다
  • `>=`: 크거나 같다
  • `<=`: 작거나 같다
  • `==`: 같다 (⚠️ 등호 두 개! `=`는 값을 넣는 것!)
  • `!=`: 같지 않다

예시:

`a == 9` 의 결과: True

여러 조건을 조합하여 논리적인 판단을 할 때 사용하며, 결과는 참(True) 또는 거짓(False)입니다.

  • `and`: 그리고 (모든 조건이 참일 때만 True)
  • `or`: 또는 (하나라도 조건이 참이면 True)

"논리는 일상에서도 쓰는 거랑 똑같아요~ 조건문 if에 들어가면 여기부터 쓰게 돼요~"

예시:

`a = 9`, `b = 4` 일 때

`a > 10 and b > 3` 의 결과: False

`a < 10 or b < 2` 의 결과: True

변수 활용 심화: 문자열 연산

변수에는 숫자만 넣는 것이 아닙니다! 문자(텍스트)도 넣고 연산할 수 있어요.

"얘들아~ 변수에 숫자만 넣는 줄 알았지? 문자도 넣고 더하고 곱하고 다 된다~"

문자열 연산의 종류

  • `+`: 문자열 이어붙이기 (Concatenation)

    예: `"안녕" + "하세요"` → `"안녕하세요"`

  • `*`: 문자열 반복하기 (Repetition)

    예: `"안녕" * 3` → `"안녕안녕안녕"`

"문자 * 숫자? 돼요! 세 번 반복되게! 근데 문자 + 숫자는 안 돼요~ 에러나요!"

복합 대입 연산자

변수 값에 연산을 더해서 다시 그 변수에 저장하는 편리한 방식입니다.

  • `a += 3` → `a = a + 3` 와 같습니다.
  • `b *= 2` → `b = b * 2` 와 같습니다.

"얘들아, 이건 실기 문제로도 나와요! '복합'이란 단어 나오면 이거예요~"

표준 입출력 (Input/Output)

사용자로부터 입력을 받고, 결과를 화면에 출력하는 방법을 배워봅시다.

입력: `input()` 함수

`input()` 함수를 사용하여 사용자로부터 값을 입력받을 수 있습니다. **주의할 점은 `input()`으로 입력받은 값은 항상 문자(str) 형태로 받아진다는 것입니다.**

  • `a = input("이름을 입력하세요: ")`
  • `num = input("숫자를 입력하세요: ")` (※ 이 상태로 `num`을 숫자로 계산하면 오류!)

"input은 다 문자야! 정수 계산하려면 꼭 int()로 싸줘야 돼요~"

형변환 함수

`input()`으로 받은 문자열을 숫자로 계산하고 싶을 때는 '형변환'을 해야 합니다.

  • `int()`: 문자열을 정수형(Integer)으로 변환합니다. (예: `int("123")` → `123`)
  • `float()`: 문자열을 실수형(Float)으로 변환합니다. (예: `float("3.14")` → `3.14`)

출력: `print()` 함수

`print()` 함수를 사용하여 프로그램의 결과를 화면에 출력합니다. 여러 값을 출력할 때는 콤마(`,`)로 구분합니다.

"문자랑 숫자랑 같이 출력할 땐 콤마로 구분해줘요! + 쓰면 에러 나요!"

파일 입출력 기초

디스크에 저장된 파일을 읽고 쓰는 방식, 영구적인 데이터 저장의 시작입니다.

지금까지는 프로그램이 종료되면 데이터가 사라지는 '표준 입출력(키보드/모니터)'을 사용했습니다. '파일 입출력'은 데이터를 하드 디스크 같은 저장 장치에 영구적으로 저장하고, 필요할 때 다시 읽어오는 방식입니다.

"파일은 읽고 쓰고 닫아야 돼요. 까먹으면 저장이 안 돼요!"

파일 입출력의 3단계

  1. **파일 열기 (`open()`):** 어떤 파일을 어떤 목적으로(읽기/쓰기/추가) 사용할지 지정합니다.
  2. **작업 (`write()`, `read()`, `readline()`, `readlines()` 등):** 파일에 내용을 쓰거나, 파일에서 내용을 읽어옵니다.
  3. **파일 닫기 (`close()`):** 작업이 끝난 파일을 안전하게 닫습니다. (매우 중요!)

주요 파일 관련 함수

`f = open("파일명.txt", "모드")`

파일을 열고 파일 객체를 반환합니다.

  • `"r"`: 읽기 모드 (Read) - 파일이 없으면 오류 발생
  • `"w"`: 쓰기 모드 (Write) - 파일이 없으면 새로 생성, 있으면 기존 내용 덮어씀
  • `"a"`: 추가 모드 (Append) - 파일이 없으면 새로 생성, 있으면 기존 내용 뒤에 추가

`f.write("내용")`

파일에 문자열을 씁니다. `print()`와 비슷하지만 대상이 모니터가 아닌 파일입니다.

"write는 파일에 적는 거예요. print랑 비슷하지만 대상이 파일임~"

`f.close()`

열었던 파일을 닫습니다. 변경사항 저장 및 자원 해제를 위해 필수입니다.

`f.read()`

파일 전체 내용을 하나의 문자열로 읽어옵니다.

`f.readline()`

파일에서 한 줄씩 읽어옵니다.

`f.readlines()`

파일의 모든 줄을 읽어와 리스트 형태로 반환합니다.
