1. 변수 활용 심화 및 문자열 연산

이 섹션에서는 변수가 숫자뿐만 아니라 문자(문자열)도 저장할 수 있다는 것을 배우고, 문자열끼리 어떻게 더하고 곱하는지 직접 실험해 봅니다. '안녕'과 '하세요'가 만나면 어떻게 될까요?

문자열 연산 시뮬레이터

실행 결과 및 코드

a = "나는 "
b = "바보입니다."

# 이곳에 실행 코드가 나타납니다.

선생님: "문자 * 숫자? 돼요! 세 번 반복되게! 근데 문자 + 숫자는 안 돼요~ 에러나요!"

복합 대입 연산자 (`+=`, `*=`)

변수 값에 연산을 더해서 다시 변수에 저장하는 방식! 실기 시험에도 자주 나와요~

a += 3 # a = a + 3
b *= 2 # b = b * 2

2. 표준 입출력 및 형변환

사용자로부터 입력을 받고(input), 결과를 화면에 보여주는(print) 방법을 배웁니다. 특히, `input()`으로 받은 값은 항상 '문자열'이라는 점! 숫자로 계산하려면 반드시 '형변환'이 필요하다는 것을 기억해야 해요.

선생님 말씀! 📢

"input은 다 문자야! 정수 계산하려면 꼭 int()로 싸줘야 돼요~ 문자랑 숫자랑 같이 출력할 땐 콤마(,)로 구분해줘요! + 쓰면 에러 나요!"

입력 및 출력 시뮬레이터

실행 코드 및 출력

name = input("이름을 입력하세요: ")
age = int(input("나이를 입력하세요: "))
print("안녕하세요,", name, "님! 내년이면", age+1, "살이 되시겠네요!")

여기에 출력 결과가 표시됩니다.

`input()`은 사용자 입력 전까지 프로그램이 멈추는 '대기 상태'가 됩니다. 커서가 깜빡깜빡하면 입력 대기 중이에요!

형변환 함수:

`int()`: 정수형으로 변환
`float()`: 실수형으로 변환

3. 오늘의 핵심 요약 및 실기 팁 ✨

오늘 배운 내용을 최종적으로 정리하고, 실기 시험에 대비하기 위한 선생님의 특급 조언을 확인하세요. 이 부분은 정말 중요하니, 꼭 몸에 익히고 다음 수업에 참여해야 합니다!

핵심 요약 포인트

  • 변수에는 숫자뿐만 아니라 **문자열도 저장**할 수 있습니다.
  • 문자열도 **더하기(+), 곱하기(*) 연산**이 가능합니다. (단, 문자 + 숫자는 에러!)
  • `input()` 함수로 입력받은 값은 항상 **문자열(str) 형태**이므로, 숫자로 계산하려면 반드시 `int()`나 `float()`로 **형변환**해야 합니다.
  • `print()` 함수는 여러 값을 콤마(`,`)로 구분하여 출력할 수 있습니다. (문자열 `+` 문자열은 가능, 문자열 `+` 숫자는 에러!)
