3. 선택 구조 (if) 실습하기

미세먼지 농도에 따라 다른 메시지를 보여주는 예제를 통해 `if`, `elif`, `else`의 쓰임새를 직접 확인해 보세요.

★ 조건 분기

  • ~30: 좋음
  • 31~80: 보통
  • 81~150: 나쁨
  • 151~: 매우 나쁨

실행 결과

# Python Code
dust = 80
if dust <= 30:
    print("좋음")
elif dust <= 80:
    print("보통")
elif dust <= 150:
    print("나쁨")
else:
    print("매우 나쁨")

# 출력
> 보통

🚦 break vs continue

break 예제 (3을 만나면 중단)

  if i == 3:
    break
